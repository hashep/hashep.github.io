안녕하세요! 컴퓨터공학과에 재학 중인 임우성입니다.

아직 실무 경험은 없지만, 다양한 팀 프로젝트와 과제를 수행하며 문제 해결 능력과 협업 역량을 꾸준히 키워왔습니다.

그 과정에서 다음과 같은 프로젝트들을 경험했습니다:

  • 자연어 기반 스팸 텍스트 필터링 시스템 제작
  • 위성 이미지 학습을 통한 이미지 분류 프로젝트
  • 감정 분석 챗봇 개발 (Streamlit + GoEmotions)
  • PDF 기반 Q&A 챗봇 구현 (Streamlit + FAISS)
  • 정보보안 및 네트워크 구조 관련 보고서 작성
  • Steam API 기반 할인 알림 Discord 봇 개발
  • 자기소개 웹사이트 제작 (GitHub Pages 활용)
  • UE5 기반 VR 좀비 게임 개발 및 여러 소규모 게임 프로젝트

이 모든 작업은 GitHub 리포지터리에 기록하며 관리해왔고,

진행 과정마다 직접 발표 자료를 구성(PPT)하고 발표하는 역할도 맡았습니다.

실제 업무에 투입된다면 여전히 배우고 적응해야 할 부분이 많겠지만,

이러한 다양한 프로젝트 경험을 통해 빠르게 적응하고, 팀원들과 원활히 소통할 수 있는 역량을 갖추게 되었습니다.

저는 이러한 실행력과 커뮤니케이션 능력을 저의 가장 큰 강점으로 생각합니다. 감사합니다!
